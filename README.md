# SettleMate

SettleMate je mobilna aplikacija za enostavno razdelitev stroškov med skupino ljudi. Deluje podobno kot aplikacija Splitwise in omogoča hitro beleženje skupnih stroškov, izračun dolgov ter poenostavitev plačil med uporabniki.

Aplikacija je posebej uporabna pri skupnih aktivnostih, kot so potovanja, druženja ali skupno bivanje, kjer več ljudi plačuje skupne stroške.

## Delovanje
Uporabniki lahko v aplikacijo vnesejo posamezen strošek in izberejo osebe, med katere se strošek razdeli. Za pogostejše sodelovanje je možno ustvariti skupine, kjer isti uporabniki dodajajo stroške skozi daljše obdobje (npr. med potovanjem).

Za dodani strošek se določi, kdo ga je plačal, kolikšen je in kako se razdeli med preostale ljudi. Strošek je mogoče razdeliti enakomerno med vse uporabnike, ali posamično, kjer ima vsak uporabnik svoj delež. 

Aplikacija na podlagi vnesenih podatkov izračuna trenutno stanje dolgov. Nudi tudi poenostavitev dolgov med člani skupine, s čimer zmanjša število potrebnih nakazil med uporabniki.

## Funkcionalnosti
- ustvarjanje skupin uporabnikov
- vnos in urejanje stroškov
- delitev stroška med uporabnike
- pregled trenutnega stanja dolgov
- optimizacija dolgov (minimalno število nakazil)

Projekt je implementiran v programskem jeziku Rust.
